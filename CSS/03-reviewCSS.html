1. text-transform: 转换单词的大小写
  2. text-align: 设置行内元素 => 生效
    • justify: 两端对齐,多行才有效。但是最后一行不生效
    • text-align-last: 可以设置最后一行两端对齐
  3. link 和 @import 的区别： 后者会阻塞
  4. word-spacing: 单词间距
  5. 百分比: 每个元素 refer(参考) 的不同,MDN看
  6. font-family: 加引号的区分：多个单词， unicode编码的
  7. font-style: italic 文字斜体(用字体设计时的专有斜体)。 oblique:倾斜（不管有没有专门的，都倾斜）
  8. font-variant: 大小字母以小写的高度展示
  9. line-height: 两行文字基线(baseline)之间的间距
    • top
    • middle
    • baseline
    • bottom
    • 上一行的bottom到下一行顶线top的距离,就是行距
    • 纯数字时是font-size的倍数
  10. font: 【font-weight || font-variant || font-style】 font-size/line-height font-family
    • 前三个顺序无所谓，可以省略
    • line-height 可以省略。 写则必须在font-size后边
    • font-size、 font-family 不可以调换顺序，不可以省略
  11. [att]:拥有某一个属性
  12. [att=val]: 属性等于某个值
  13. 目标伪类：:target, http://xxxx/xxx/xx#demo, id为demo时
  14. 否定伪类：:not(xxx)
  15. 伪类顺序： link,visited,focus,hover,active => love fuck hate
  16. 伪元素：::first-line ::first-letter
  17. 伪元素：::after ::before, content: '', 不能省略,默认生成的是行内元素
