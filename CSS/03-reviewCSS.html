  43. 弧度和度数间转换
    • 一个完整的圆的弧度是 2π, 所以 2π rad = 360º
    • 1π rad = 180º, 1º = π/180rag; 1 rad = 180º/π(约 57.29577051º)
    • 以度数展示角度, 把数字乘以 π/180 便转换成弧度 π/180: rad = (π / 180) * deg

  46. 字节面试：什么是可替换元素？
    • 其内容是可以被替换的元素。 img 标签没有写入任何内容，其内容来自浏览器去下载 src 属性给到的图片，并用该图片资源替换掉 img 标签，在浏览器下载前并不知道图片的宽高。 所以，可替换元素比较特殊，它的宽高是由其加载的内容决定的
    • 典型的可替换元素有,iframe, video, embed, img
    • input 在特定情况下被作为可替换元素处理
