1. text-transform: 转换单词的大小写
  2. text-align: 设置行内元素 => 生效
    • justify: 两端对齐,多行才有效。但是最后一行不生效
    • text-align-last: 可以设置最后一行两端对齐
  3. link 和 @import 的区别： 后者会阻塞
  4. word-spacing: 单词间距
  5. 百分比: 每个元素 refer(参考) 的不同,MDN看
  6. font-family: 加引号的区分：多个单词， unicode编码的
  7. font-style: italic 文字斜体(用字体设计时的专有斜体)。 oblique:倾斜（不管有没有专门的，都倾斜）
  8. font-variant: 大小字母以小写的高度展示
  9. line-height: 两行文字基线(baseline)之间的间距
    • top
    • middle
    • baseline
    • bottom
    • 上一行的bottom到下一行顶线top的距离,就是行距
    • 纯数字时是font-size的倍数
  10. font: 【font-weight || font-variant || font-style】 font-size/line-height font-family
    • 前三个顺序无所谓，可以省略
    • line-height 可以省略。 写则必须在font-size后边
    • font-size、 font-family 不可以调换顺序，不可以省略
  11. [att]:拥有某一个属性
  12. [att=val]: 属性等于某个值
  13. 目标伪类：:target, http://xxxx/xxx/xx#demo, id为demo时
  14. 否定伪类：:not(xxx)
  15. 伪类顺序： link,visited,focus,hover,active => love fuck hate
  16. 伪元素：::first-line ::first-letter
  17. 伪元素：::after ::before, content: '', 不能省略,默认生成的是行内元素
  18. 继承: inherited 可以强制声明属性继承
  19. 权重:伪元素和普通元素一样
  20. css属性:权重优先级、style中先后顺序
  21. 行内替换元素:可以设置宽高, img, video, iframe
  22. p元素内不能包括div元素
  23. 元素隐藏的方法：
    • 不占空间: display:none
    • 占据空间: visibility: hidden
    • rgba: 不影响子元素
    • opacity: 影响所有的子元素(应用于一个元素上,整体,包括其所有的子元素)
  24. 块级,行内元素: 宽度是 auto, 由浏览器控制
  25. border: 梯形实现
    i am content
  26. border-style: 不可以省略
  27. border-radius: 百分比(%)时,基于当前元素的宽高来设置
  28. border-top-left-radius: x y 单独设置
  29. border-color: 会盖在background-color上, 默认会继承 color
  30. margin: 更倾向于兄弟间间隔
    • 嵌套时,左右如果和父级重叠，不会穿透
    • 1.父级设置border
    • 2.父级设置padding
    • 3.触发父级的BFC: 如 overflow: hidden
    • 4.margin-bottom: 如果父级的height:auto, 同时底部重叠, 会触发margin穿透
  31. padding: 针对于父子间间隔处理
  32. 行内元素：
    • padding 设置时, 上下看起来撑开, 但是不占空间
    • border 设置时, 上下看起来撑开, 但是不占空间
    • margin 设置时, 上下不生效
  33. block 元素: 默认宽度计算 width + padding + border + margin
  34. box-shadow: x y blur spread color inset
  35. text-shadow: x y blur color
  36. 盒模型
    • border-box: padding + border + width
    • conten-box: 只有 width, padding 和 border 在外边
    • IE盒子模型: border-box
    • 标准盒模型: width === content
  37. background-image: 设置多张图片, 其他图片按顺序层叠在下面
  38. background-attachment: 决定背景图像的位置是在视口内固定, 或者随着包含它的区块滚动
  39. background 简写: background-size 可以省略, 如果不省略, /background-size 必须紧跟在 background-position 的后面
  40. 父级 display: flex; 子元素 flex: 1, width: 0, 以flex为准。
  41. 父级 display: flex; 子元素 width: 30%, 如何实现一个正方向? 高度 height: 30%
