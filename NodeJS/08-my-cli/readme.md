# 快速部署 Vue3 项目工具

集成列表

- Vue3
- Vite2
- Pinia
- VueRouter4
- ElementPlus
- Eslint

## 安装

npm i -g vunbo-util

## 操作指令

版本查询

```js
vunbo - v
```

创建项目，并拉取基础模板（后续提供自定义模板地址）

```js
vunbo create 项目名
```

快速添加 vue 组件模板.（默认 src/components）

```js
vunbo addcomp 组件名称 -d 路径
```

快速添加 vue 组件页面.(默认 src/pages)

```js
vunbo addpage 页面名称 -d 路径
```

快速添加 vuex 模板(默认 src/store)

```js
vunbo addstore store名称 -d 路径
```
